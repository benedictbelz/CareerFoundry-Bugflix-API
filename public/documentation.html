Bugflix – API Documentation

The well known movie database. One of the best, except a couple of bugs from time to time.

Request URL Query Parameter HTTP Method Request Body Response Body
Return a list of all movies. /movies – GET – A JSON holding data about all movies. 
{
							    movies: ['Apokalypse Now','Inception','Children of Men']
							}
Return data about a single movie by title. /movies/[name] /movies/:name GET – A JSON holding data about the movie. 
{
							    name: 'Apokalypse Now',
							    descripton: 'A U.S. Army officer serving in Vietnam is tasked with assassinating a renegade Special Forces Colonel who sees himself as a god.',
							    genres: ['Drama','Mystery','War'],
							    director: 'Francis Ford Coppola',
							    image URL: 'apokalypsenow.png'
							}
Return data about a genre by title. /genres/[name] /genres/:name GET – A JSON holding data about the genre. 
{
							    name: 'Drama',
							    descripton: 'Drama is a category of narrative fiction intended to be more serious than humorous in tone.'
							}
Return data about a director. /directors/[name] /directors/:name GET – A JSON holding data about the director.
Allow new users to register. /users – POST A JSON object holding data about the user to create. A text message that the user has been successfully created.
Update user information. /users/[name] /users/:name PUT A JSON object holding data about the updated user. A JSON object holding data about the updated user.
Unregister an existing user. /users/[name] /users/:name/ DELETE – A text message that the user has been successfully removed.
Add a movie to favourites. /users/[name]/favourites/[movie] /users/:name/favourites/:movie POST – A JSON object holding data about the current favourites list.
Remove a movie from favourites. /users/[name]/favourites/[movie] /users/:name/favourites/:movie DELETE – A JSON object holding data about the current favourites list.
Get a teapot. /teapot GET – A text message with status 418.
